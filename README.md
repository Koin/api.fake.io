# api.fake.io

It's a simple service which permits you to mock a RESTful web service.

## System dependencies 

* ruby (1.9.3)
* rails (4.0.0.beta)
* sqlite3 (>= 3.7.12)

## Install

### Dependencies
```sh
$ bundle
```

### Database creation
```sh
$ rake db:migrate
```
